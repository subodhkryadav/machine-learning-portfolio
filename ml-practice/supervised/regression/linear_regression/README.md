# Linear Regression

This project demonstrates the implementation of the
Linear Regression algorithm using Python.

## Objective
To train and evaluate a Linear Regression model on a given dataset
and understand the relationship between input features and target values.

## Files
- linear_regression.ipynb – Model implementation
- dataset.csv – Dataset used
- requirements.txt – Required libraries

## Workflow
1. Load dataset
2. Preprocess data
3. Train model
4. Make predictions
5. Evaluate performance

## Libraries Used
Python, NumPy, Pandas, Matplotlib, Scikit-learn
