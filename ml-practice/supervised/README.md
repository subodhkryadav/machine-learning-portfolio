## üìà 2. Regression Module (8 Algorithms)
This section includes models used to predict continuous numerical values (Excludes Logistic & Naive Bayes).

1. **Linear Regression** - Fundamental continuous prediction.
2. **K-Nearest Neighbors (KNN) Regressor** - Proximity-based value prediction.
3. **Support Vector Regression (SVR)** - Epsilon-tube based fitting.
4. **Decision Tree Regressor** - Regional average partitioning.
5. **Random Forest Regressor** - Ensemble tree averaging.
6. **AdaBoost Regressor** - Boosting for numerical tasks.
7. **Gradient Boosting Regressor** - Residual error correction.
8. **XGBoost Regressor** - High-performance regression with **Optuna**.
